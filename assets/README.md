# Assets (source originals)

Put **source** images, SVGs, or brand files here if you like keeping them next to the repo.

**For the live site**, place files you want served at a URL under **`/public`** (e.g. `public/assets/photo.jpg` → use `src="/assets/photo.jpg"` in `portfolio.ts`).

Next.js does not serve files from this `assets/` folder by default.
